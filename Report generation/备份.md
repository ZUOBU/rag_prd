基金分析报告生成PRD（股票型、混合型-偏股）（单支）
本功能旨在通过AI自动调取数据库中的数据，结合大模型的语义理解能力，为客户（及客户经理）提供一份专业、直观、且具备个性化持仓建议的基金分析报告。

# 一、产品概述

| 项目   | 内容                  |
| ---- | ------------------- |
| 产品名称 | 基金分析报告生成器           |
| 目标用户 | 零售客户 / 客户经理 / 投顾    |
| 核心能力 | 单基金分析 & 双基金对比分析     |
| 输入   | 基金代码（最多2只）、客户ID（可选） |
| 输出   | 结构化 + 解释性基金分析报告     |
| 核心原则 | 数据驱动 + LLM总结 + 过程透明 |

# 二、整体流程设计（含过程展示）

### 1. 报告生成流程（对用户可见）

| 步骤序号 | 步骤名称   | 展示文案（前端）    | 处理内容       |
| ---- | ------ | ----------- | ---------- |
| 1    | 基金信息校验 | 正在获取基金基础信息  | 校验基金代码、类型  |
| 2    | 数据加载   | 正在加载市场与基金数据 | 查询数据库      |
| 3    | 指标计算   | 正在计算收益与风险指标 | Sharpe、回撤等 |
| 4    | 持仓分析   | 正在解析持仓结构    | 行业/股票分布    |
| 5    | 估值分析   | 正在评估基金估值水平  | PE/PB分位    |
| 6    | 基金经理分析 | 正在分析基金经理能力  | 任期、历史收益    |
| 7    | 智能总结   | 正在生成分析结论    | LLM生成      |
| 8    | 报告生成   | 正在生成完整报告    | 汇总输出       |

# 三、报告结构设计

## 1、简短评价

| 项目       | 设计                                                                  |
| -------- | ------------------------------------------------------------------- |
| 生成方式       | LLM生成                                                               |
| 输入数据     | <ul><li>基金名称</li><li>基金代码</li><li>近1年收益涨跌</li><li>近半年收益涨跌</li><li>近3月收益涨跌</li><li>最大回撤</li><li>波动率</li><li>基金类型</li><li>比较基准近一年收益涨跌</li></ul> |
| Prompt要点 | “请用一句话总结该基金的收益与风险特征，并给出评级（优秀/中等/偏弱）”                                |
| 输出形式     | 150字以内的核心定性结论                                                               |
| 示例       | “你持有的永赢科技智选混合A(022364) 近期出现小幅回调，但长期收益表现突出，跑赢了同类基金。这只基金属于高风险高收益的科技主题产品，需要结合你的持仓结构综合评估。”                                       |


## 2、智能摘要



| 子模块     | 数据来源            | 生成方式 | 说明      |
| ------- | --------------- | ----- | ------- |
| 配置建议摘要  | 客户风险等级 + 基金风险等级 + 综合评估| LLM生成     | 解释是否建议配置 |
| 产品表现    | 收益率、排名、回撤       | LLM生成 | 根据数据作出总结（不展示数据本身）   |
| 行业分析    | 持仓行业分布          | LLM生成 | 评估该行业长期发展趋势，以及短期波动情况 |
| 与客户持仓关系 | 客户持仓数据 | LLM生成 | 评估基金与客户持仓的重合度/分散性 |

### LLM输入结构（示例）

```json
{
  "fund_return_rank": "前20%",
  "max_drawdown": "-15%",
  "industry_exposure": ["科技40%", "消费20%"],
  "user_portfolio_overlap": "30%",
  "risk_level_match": "匹配"
}
```

# 四、双基金对比设计（重点）

| ---- | --------- |
| 收益能力 | 并列展示 + 排名 |

| 行业分布 | 双柱状图      |

| 综合评价 | LLM生成     |

### LLM Prompt（对比）

```text
请比较基金A与基金B在收益、风险、行业配置上的差异，并给出适合人群。
```

# 八、关键设计亮点

1. **数据优先**：避免“幻觉”
2. **可解释性强**：每一步透明
3. **支持对比**：提升决策能力
4. **图表驱动**：增强理解
5. **客户视角**：结合持仓推荐
